# speed-mern-app-demo-frontend

Speed your MERN app by REDIS

## Project setup

### (1) Config (optional)

Please create **.env file** at root and add appropriate "API_URL"

```js
//consider sample
REACT_APP_API_URL=http://localhost:3001/api/
```

### (2) Install packages

```sh
npm install
```

### (3) Run project locally

```sh
npm start
```
